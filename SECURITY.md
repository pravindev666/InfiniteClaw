# Security Policy

## Supported Versions

InfiniteClaw is under active development. Only the latest version on the `main` branch is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our infrastructure management platform seriously. If you discover a security vulnerability within InfiniteClaw, please do not open a public issue.

Instead, please report the vulnerability by contacting the lead maintainer directly or using the GitHub Security Advisory feature.

## Cyber-Security Philosophy
InfiniteClaw is designed as an autonomous "Alpha-Mode" engine. High-risk execution capabilities are inherently powerful. We rely on community audits and radical transparency.

Review our official [Disclaimer](DISCLAIMER.md) for more details on user responsibility.
